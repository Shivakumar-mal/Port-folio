My Work

Diagnosis of Dementia Using Eye Movements

In my project on diagnosing dementia using eye movements, I developed a non-invasive method to detect dementia through machine learning. Using eye-tracking data, I applied various models, including SVM, Random Forest, and Logistic Regression, with Logistic Regression achieving the highest accuracy. This innovative approach demonstrated the potential for early and accurate dementia detection. Throughout this project, I utilized Python, eye trackers, and advanced data processing techniques, showcasing my skills in machine learning and data analysis to improve healthcare

Vehicle Accident Prevention on Mountain Roads.

The project "Vehicle Accident Prevention on Mountain Roads" aims to improve safety on treacherous mountain roads, known for their steep inclines and sharp turns. By employing advanced technologies such as ultrasonic sensors, LEDs, and a camera module, the system provides real-time data to prevent accidents. Key features include detecting vehicle speed and distance near curves, counting vehicles, and offering visual cues through LED indicators. A user-friendly website was developed to communicate road conditions, supported by real-time data storage via Firebase. The project successfully integrates these components to enhance road safety and offers a reliable solution to reduce accident risks on challenging terrains.

Sentiment Analysis-Based Harassment Detection using Machine Learning

In the project "Sentiment Analysis-Based Harassment Detection using Machine Learning," I developed an automated system to identify online harassment in real-time. The goal was to enhance online safety by leveraging sentiment analysis and supervised learning algorithms. Using techniques like TF-IDF and CounterVectorizer for feature extraction, and Support Vector Machines (SVM) for classification, the system achieved high accuracy with minimal false positives. This project utilized Python, NLTK, scikit-learn, and pandas to create an effective solution for detecting and flagging harassment on social media platforms.
